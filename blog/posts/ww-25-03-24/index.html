WorldWise - Enhancing the Learning Experience! Iteration 1+

25 Mar 2024

Welcome back to the WorldWise DevBlog! After a little more than two weeks, I’m excited to share the progress I’ve made on the app during Iteration 1 and beyond. Since my last post, I’ve been hard at work refining the app’s functionality and user experience. Here’s a glimpse into what I’ve been up to:

User Interface Improvements

  • Simplified Navigation: I’ve replaced the ‘Skip’ button with an ‘I Don’t Know’ and ‘Next’ system, streamlining the learning process and making it more intuitive.
  • Exercise Redesign: The exercises have undergone a major design overhaul, improving both aesthetics and usability.

Exercise Enhancements

  • Validation System: Exercises now feature a ‘Validate’ button before moving on to the next question, ensuring users have the chance to confirm their answers.
  • Answer Class: I’ve implemented a new Answer class for better generalization across different types of exercises.
  • Customization: The exercise generation now better follows user customization.
  • Write Exercise: A new writing exercise type has been introduced, allowing users to practice writing the names of countries/capitals.
  • Match Exercise: A matching exercise type has been added, providing a fun and interactive way to test your knowledge.

Backend Refinements

  • Project Structure: The project has been refactored into individual files for better maintainability and scalability.
  • History Widget: The history feature has been converted into its own widget, and I’ve included some filters for improved data management.

User Experience

  • Exercise Categories: A new ExerciseCategory system has been introduced, allowing for a more organized development of new exercise types.
  • Feedback on Mistakes: When a mistake is made, the app now displays the correct answer and provides an explanation.

Bug Fixes and Updates

  • Discovery Repetition: Fixed an issue where the discovery exercise was repeating the same country.

What’s Next?

I’m not stopping here! My next steps include:

  • Prediction: I need to refine the prediction of the knowledge retention as it is currently not as accurate as I would like.
  • Difficulty: I will start to work on the difficulty adaptation system to ensure it is tailored to each user’s learning pace.
  • Learning Style: I plan to implement a learning style adaptation system to provide exercises that align with the user’s preferred learning methods.
  • Intuder Exercise: I will introduce a new type of exercise where users must identify the incorrect information among a set of options.
  • Initial Knowledge: I will work on a system to assess the user’s initial knowledge to provide a more personalized learning experience.
  • Hint System: I will implement a hint system to guide users when they are stuck on an exercise.
  • Customization: I will continue to refine the customization options to give users more control over their learning experience.

Stay tuned for more updates, and thank you for joining me on this exciting journey with WorldWise!
