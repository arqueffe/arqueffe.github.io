Multi-Agent Path Finding

Decision Problem:

Input: A graph $G = \langle V, E \rangle$ with $V$ a finite set of nodes and $E$ a set of movement edges, an initial and a goal configuration $c^i$, $c^g$ elements of $V^n$, with $n$ the number of agents.

Output: Does there exist an execution to reach $c^g$ from $c^i$ without collision ?

Optimization Problem:

Input: A graph $G = \langle V, E \rangle$ with $V$ a finite set of nodes and $E$ a set of movement edges, an initial and a goal configuration $c^i$, $c^g$ elements of $V^n$, with $n$ the number of agents, and an integer $k$.

Output: Does there exist an execution of cost $k$ to reach $c^g$ from $c^i$ without collision ?

Connectivity

Connected MAPF

  • Introduced and shown NP-hard by Hollinger and Singh [69]
  • Existence of an execution shown PSPACE-hard by Tateo et al. [151]

Decision Problem:

Input: A topological graph $G = \langle V, E^m, E^c \rangle$ with $V$ a finite set of nodes and $E^m$ a set of movement edges, $E^c$ a set of communication edges, an initial and a goal configuration $c^i$, $c^g$ elements of $V^n$, with $n$ the number of agents.

Output: Does there exist a connected execution to reach $c^g$ from $c^i$ ?

Optimization Problem:

Input: A topological graph $G = \langle V, E^m, E^c \rangle$ with $V$ a finite set of nodes and $E^m$ a set of movement edges, $E^c$ a set of communication edges, an initial and a goal configuration $c^i$, $c^g$ elements of $V^n$, with $n$ the number of agents and an integer $k$.

Output: Does there exist a connected execution of cost $k$ to reach $c^g$ from $c^i$ ?

Connected Configuration: A configuration $c$ of size $n$ is connected if and only if the sub-graph $G_c=\langle V_c, E^c \rangle$ with $V_c = \{ c_1, \dots, c_n \}$ is a connected graph.

Connected Execution: An execution $exec=\langle c^1, \dots, c^k \rangle$ is connected if and only if for all $0< i\leq k$ such that $c^i$ is a connected configuration.

Directed Topological Graphs

$E^m$ and $E^c$ are arbitrary

CMAPF is PSPACE-complete and Bounded CMAPF is NP-complete.

Undirected Topological Graphs

$E^m$ is undirected

CMAPF is PSPACE-complete and Bounded CMAPF is NP-complete.

Neighbor-Communicable Topological Graphs

$E^m \subseteq E^c$

CMAPF is PSPACE-complete and Bounded CMAPF is NP-complete.

Sight-Moveable Topological Graphs

CMAPF is in LOGSPACE and Bounded CMAPF is NP-complete.

Complete-Communication Topological Graphs

$E^c = V \times V$

CMAPF is in LOGSPACE and Bounded CMAPF is NLOGSPACE.

Low-Level

Computes the path of an agent given a set of constraints.

CMAPF with Incomplete Knowledge

Initial Knowledge and Discovery

Input: Two topological graph $G_1$ "must", $\mathbf{G_2}$ "may", an initial and a goal configuration $c^i$, $c^g$ elements of $V^n$, with $n$ the number of agents.

Output: Does there exist a connected joint strategy to reach $c^g$ from $c^i$ ?

Output: Does there exist a decentralized joint strategy to reach $c^g$ from $c^i$ ?

Strategy: A strategy $\sigma_i$ for agent $i$ maps any execution $\pi$ to a vertex such that $(last(\pi)_i,\sigma_i(\pi))\in E^m$.

Uniform: A strategy $\sigma_i$ is uniform if for all $|\pi|=|\pi'|$ with identical knowledge for agent $i$, $\sigma_i(\pi) = \sigma_i(\pi')$.

Connected: A strategy $\sigma$ is connected if all outcomes are connected.

Decentralized: A strategy $\sigma$ is decentralized if agents disconnect and share their knowledge upon reconnection.

Bounded: A strategy $\sigma$ is bounded by $k$ if all outcomes reaches the goal in $k$ steps.
